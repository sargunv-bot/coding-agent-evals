# Qualitative review

After deterministic verification, a separate SOTA reviewer assesses the committed patch and trajectory for:

- scope discipline;
- code clarity;
- test quality;
- repository fit and taste;
- security and safety;
- mergeability and explicit blockers.

Scores are 1–5 evidence labels, not a substitute for deterministic correctness. The review records the reviewer identity and model and is not model-blind in the current campaign. A deterministic pass can still be judged unmergeable; a deterministic failure cannot be promoted to a behavioral pass by subjective review.
